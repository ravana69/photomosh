Upgrade your Glitching with PhotoMosh-Pro.

Unlimited creative control for image and video distortion.

PhotoMosh-Pro is a fully featured desktop version of PhotoMosh, including new and improved effects, high-res video exports, improved UI, modulators, audio-reactive effects and more.

We're obsessed with creating powerful tools with simple interfaces. Our goal is to inspire you and help your work to flow.

New Effects.
Over 20 brand new effects, in addition to all the effects on photomosh.com. Stack effects for infinite combinations. Designed for maximum image and video creativity.
Luma-Mesh
3D luminance depth distortion Light grid / RuttEtra Effect. Control luminance height, 3D tilt motion, mesh opacity, bands and warp height.
Optical-Flow
Motion reactive liquid optical flow. Great for high motion input.
Data-Mosh
Motion reactive glitching simulating video p-frame dropping.
Feedback
Temporal feedback. Motion trails. Control scale, rotation, warp and hue shift.
Soft Glitch
Soft gradient chromatic aberration glitching.
Hard Glitch
Chunky hard multi-scale randomized displacement. Control amount, scale, color split and speed.
Stretch
Stretch out area of input with optional wave and jaggies.
8-Bit CGA
CGA style pixelation for that 1982 old school vibe. 3 preset color schemes.
Kaleidoscope
The VJ's favorite. Make anything look cool. Control number of sides and angle.
Light Streak
Get glossy with smooth randomized light streaks. Tweak angle and strength.
VHS
Classic 80's vibes. Optional time display, static and bar smearing.
Super 8
70's slideshow effect with optional dust, burn, vignette framing and shutter roll.
Overlay
Load any image file to use as an overlay above other effects, using alpha transparency or additive blend. Stack multiple overlays.
Mask Draw and Mask Blocks
Mask Effects allow you to mask out individual effects either by drawing on screen or with animated block masks.
Decimate
Multi-Pass glitch giving some nice complex shapes.
Tile
Sliding mirrored tiling. Control size, speed and angle.

Other new effects include Mouse Push, Bulge, Grain, Strobe, Transform, Color Correction and more. We have also improved existing effects adding more fine grain control.

High-Res Exports.
PhotoMosh-Pro gives full control of export settings. Control output size, duration, frame rate and file type.
  • Output video up to 4K resolution with unlimited duration. (Render time dependent on device capabilities.)
  • Video Output to MP4 or GIF. Image Output to JPG or PNG.
  • Transparency / Alpha Channel support for PNG & GIF (input and output).
  • Handles GIF input. Supported input file types are: MP4, MOV, WEBM, GIF, PNG & JPG.
  • Control scene background color and opacity.
  • Toggle audio output.
  • Save at up to 60 FPS (frames per second).
  • Export render size limited to 4096 x 4096 pixels or smaller.
Make Perfect Loops.

Effect Modulators allow you to fine control effect parameters over time via various methods: linear, sine wave, looping noise, and random. Control modulator speed, multiply factor and cut-off chance. Gives more variation to loops by varying glitch amount over time.

All effects are now designed for perfect seamless looping.

Audio Reactive Modulators allow you to use audio input volume to drive effect parameters, creating automatic audio-reactive animations.

Rebuilt & Improved UI.

PhotoMosh-Pro UI has been rebuilt from scratch to improve performance and give increased fine grain controls over your glitching.

  • Presets allow saving and loading your favorite effect combinations.
  • Add & Remove Effects from the FX list. Allows duplicate FX stacking. Gives infinite FX combinations.
  • Draggable Effects. Switch order of effects by dragging.
  • Video Playback controls. Pause/Play and skip input video.
  • Pop-Out Preview window allows an easy way to project at full screen or stream and record via OBS.
  • Standalone desktop app for Windows & Mac.
  • No internet connection required.
  • Cleaner, more intuitive UI.
